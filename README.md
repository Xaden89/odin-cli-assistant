# Odin CLI Assistant

Odin is a simple command-line personal assistant written in Python.  
It is designed as a learning project to explore Python fundamentals, file-based memory, and basic conversational logic.

## Features
- Interactive CLI interface
- Remembers simple facts using a JSON file
- Answers basic questions about the user
- Designed to be extended as skills improve

## Requirements
- Python 3.9 or newer

## Usage
Run Odin from the terminal:

```bash
python3 odin.py
